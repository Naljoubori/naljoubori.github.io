Web design is the process of planning, conceptualizing, and arranging content intended for the Internet. Modern web design goes beyond how things look (aesthetics) to include how things work (functionality).

   WEB DESIGN TIPS

Failing to plan is planning to fail. This is also true in web design. When the basics are in place, will you be able to put creative touches on top. Here’s what planning can look like: Define the website’s goal, Sketch out the vistor’s journey, Come up with a style guide, Focus on the Structure, Stick With Web Standards for consistency, Attention to Content Formatting, Try with one page first, and A/B testing.
